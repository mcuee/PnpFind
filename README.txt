pnpfind Vista/7 driver storage managment console program by Travis Robinson

This will only work for Windows Vista, Windows Server 2008, 
Windows 7 and Windows Server 2008 R2 or later. You need to 
run as admin if you want to use /delete command to remove
driver packages. 

Please do not use this under Windows XP or lower. 

Source is also included.

Similar utilites:
https://github.com/lostindark/DriverStoreExplorer
https://github.com/JustAMan/pyWinClobber/blob/master/driver_cleanup.py

Usage:
C:\work\pnpfind\PnpFind>PnpFind /?
